# QR Code Generator

A simple QR code generator web app. Type any text or URL to generate a QR code, or leave the input empty to generate a random GUID.

**Live at:** https://qr.mateovandamme.com/

![Screenshot](./example.png)

## Features

- Text input field for custom text/URLs
- Random GUID generation when no input is provided
- Mobile-friendly, dark theme UI
- Uses [nayuki/QR-Code-generator](https://github.com/nayuki/QR-Code-generator) for efficient encoding (automatic alphanumeric/numeric mode optimization)
